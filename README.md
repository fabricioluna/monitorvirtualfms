O portal **Luna MedClass** é uma plataforma de suporte acadêmico desenvolvida para estudantes de Medicina. O foco é o aprendizado colaborativo e a prática através de simulados avançados com IA e ferramentas de gestão acadêmica.

## 🔗 Link do Repositório
*(Nota: Lembre-se de atualizar este link quando renomear o repositório no GitHub)*
[https://github.com/fabricioluna/lunamedclass](https://github.com/fabricioluna/lunamedclass)

## 🚀 Principais Funcionalidades

- **🧠 Simulados de IA**: Questões dinâmicas usando o modelo Gemini.
- **🩺 Estações OSCE**: Simulador de estações clínicas com feedback "Padrão-Ouro".
- **📊 Calculadoras Acadêmicas**: Cálculo automatizado de médias para UCIV, HM2, IESC e UCCG.
- **📁 Repositório Colaborativo**: Compartilhamento de resumos e roteiros práticos.
- **⭐ Quiz de Especialidade**: Teste vocacional médico detalhado.

## 🛠️ Tecnologias

- **Frontend**: React 19 (ESM) + Tailwind CSS + Lucide Icons.
- **Backend**: Firebase Realtime Database.
- **Inteligência Artificial**: Google Gemini API.

## 📦 Como Rodar
Como o projeto utiliza **ES Modules** via `importmap`, não é necessário `npm install`. Basta abrir o `index.html` usando um servidor local (ex: Live Server do VS Code).

## 🌍 Deployment
Este projeto pode ser hospedado facilmente no **GitHub Pages**, **Vercel** ou **Firebase Hosting**.
